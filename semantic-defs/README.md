# Canonical semantic definitions for The Cohesive Tetrad

This directory contains the canonical bilingual semantic definitions for the
core concepts of The Cohesive Tetrad: Sabda, Logic, Qualia, Mystica, and
Akhlak. All manuscripts, datasets, AI models, and institutional
implementations that claim to apply The Cohesive Tetrad must treat the files
in this directory as the primary semantic authority.

The canonical corpus is released under CC0 1.0 Universal (Public Domain
Dedication) and is part of the canonical record associated with the DOI
10.17605/OSF.IO/D5S7V.

---

## I. English canonical overview

### 1. Role of this directory in the TCT corpus

The Cohesive Tetrad is an epistemic and ethical framework for truth governance
built on four interdependent languages of truth:

- Sabda (Revelatory Word, the authoritative source of norm and telos)
- Logic
- Qualia
- Mystica

These four dimensions are tested and verified on the surface of Akhlak, the
relatively stable configuration of moral dispositions and governance
practices.

The files in `semantic-defs/` define the canonical semantic field of these
terms in both Indonesian and English. When there is any ambiguity or
conflict in the use of terminology, the definitions in this directory take
precedence over other descriptions in the repository or in external
documents.

### 2. Files in this directory

- `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.json`  
  Machine readable bilingual semantic kernel for Sabda, Logic, Qualia,
  Mystica, and Akhlak. Intended for programmatic use, data pipelines, and AI
  alignment workflows.

- `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.md`  
  Full canonical semantic definitions in Indonesian and English. This is the
  primary textual authority for human readers.

- `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.pdf`  
  Reading edition of the full semantic definitions for offline use and
  citation.

- `TCT_Canonical-Semantic-Definitions_Short_ID-EN_v1.0.md`  
  Short canonical semantic definitions suitable for glossaries, journal
  articles, documentation, and technical notes.

- `TCT_Canonical-Semantic-Definitions_Short_ID-EN_v1.0.pdf`  
  Reading edition of the short semantic definitions.

All files in this directory refer to the same conceptual content and are
synchronized for version 1.0.

### 3. Guidance for use by researchers and AI systems

1. For **human readers**, start from  
   `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.md`. Use the short version
   only as a compact reference, not as a replacement of the full document.

2. For **AI and software systems**, use  
   `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.json` as the primary
   semantic kernel. Keys, identifiers, and structure should be treated as
   stable within the 1.0 corpus. If you extend the schema, keep the original
   identifiers unchanged.

3. For **derivative works** such as translations, datasets, and models,
   explicitly state that the semantic source of truth is this directory and
   cite the canonical DOI 10.17605/OSF.IO/D5S7V.

4. Any proposed changes to the definitions should be discussed as versioned
   updates to this directory and recorded in the root `CHANGELOG.md` to
   preserve the audit trail for truth governance and research integrity.

## II. Ikhtisar kanonis dalam Bahasa Indonesia

### 1. Fungsi direktori dalam korpus TCT

The Cohesive Tetrad adalah kerangka epistemik dan etis untuk tata kelola
kebenaran yang dibangun di atas empat bahasa kebenaran yang saling
terjalin:

- Sabda (Revelatory Word, the authoritative source of norm and telos)
- Logika
- Qualia
- Mistika

Keempat dimensi ini diuji dan diverifikasi pada permukaan Akhlak, yaitu
konfigurasi disposisi moral dan tata kelola yang relatif stabil sehingga
berulang kali melahirkan tindakan yang amanah, adil, jujur, penuh belas
kasih, dan memuliakan martabat setiap insan.

Berkas-berkas di dalam `semantic-defs/` menetapkan ranah semantik kanonis
untuk istilah-istilah tersebut dalam bahasa Indonesia dan Inggris. Apabila
terjadi kerancuan atau perbedaan istilah dalam naskah lain, definisi di
direktori ini yang harus dijadikan rujukan utama.

### 2. Berkas yang tersedia

- `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.json`  
  Inti semantik dwibahasa yang dapat dibaca mesin untuk Sabda, Logika,
  Qualia, Mistika, dan Akhlak. Dipakai untuk pemrograman, pipeline data, dan
  penyelarasan AI.

- `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.md`  
  Naskah lengkap definisi semantik kanonis dalam bahasa Indonesia dan
  Inggris. Menjadi otoritas teks utama bagi pembaca manusia.

- `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.pdf`  
  Edisi PDF dari definisi semantik lengkap untuk pembacaan luring dan
  keperluan sitasi.

- `TCT_Canonical-Semantic-Definitions_Short_ID-EN_v1.0.md`  
  Definisi semantik kanonis versi ringkas yang cocok sebagai glosarium,
  lampiran artikel jurnal, dokumentasi, dan catatan teknis.

- `TCT_Canonical-Semantic-Definitions_Short_ID-EN_v1.0.pdf`  
  Edisi PDF dari definisi semantik ringkas.

Semua berkas ini merujuk pada isi konseptual yang sama dan telah diselaraskan
untuk versi 1.0.

### 3. Panduan penggunaan

1. Untuk **pembaca manusia**, gunakan  
   `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.md` sebagai rujukan utama
   dan manfaatkan versi ringkas hanya sebagai pengingat cepat.

2. Untuk **sistem AI dan perangkat lunak**, gunakan  
   `TCT_Canonical-Semantic-Definitions_ID-EN_v1.0.json` sebagai inti semantik
   utama. Struktur dan penanda di dalamnya perlu dijaga stabil selama masih
   berada dalam korpus versi 1.0.

3. Untuk **karya turunan** seperti terjemahan, himpunan data, dan model,
   nyatakan secara eksplisit bahwa sumber semantik adalah direktori ini dan
   cantumkan DOI kanonis 10.17605/OSF.IO/D5S7V.

4. Setiap usulan perubahan definisi harus diperlakukan sebagai pembaruan
   versi pada direktori ini dan dicatat dalam `CHANGELOG.md` di root
   repositori agar jejak audit tata kelola kebenaran dan integritas riset
   tetap terjaga.
