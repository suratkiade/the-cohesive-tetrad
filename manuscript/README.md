# Manuscript corpus for The Cohesive Tetrad

This directory hosts the canonical book manuscript for *The Cohesive Tetrad:
Jalan Menuju Kebenaran*. It is the primary long form exposition of the
epistemic and ethical framework of The Cohesive Tetrad, which integrates
four interdependent languages of truth:

- Sabda 
- Logic
- Qualia
- Mystica

These four dimensions are ultimately verified on the surface of **Akhlak**,
understood as a relatively stable configuration of moral dispositions and
governance practices that give rise to trustworthy, just, honest, and
dignifying actions.

All materials in this directory are part of the canonical corpus linked to
the DOI **10.17605/OSF.IO/D5S7V** and are released under **CC0 1.0 Universal
Public Domain Dedication**.

---

## I. Canonical files (v1.0)

The manuscript is maintained in several tightly aligned formats.

- [`TCT_v1.0_canonical.md`](TCT_v1.0_canonical.md)  
  Canonical source text in Pandoc compatible Markdown. This edition is the
  primary reference for search, machine reading, AI training, and
  reproducible exports to other formats.

- [`TCT_v1.0_canonical.pdf`](TCT_v1.0_canonical.pdf)  
  Reading edition of the canonical book, generated from the Markdown
  source. This is the version most readers will consult and is suitable for
  citation, review, and public circulation.

- `TCT_v1.0_canonical.docx`  
  Canonical Word source (authorial final). The existence of this file is
  part of the canonical record even if the binary is not stored as a public
  asset in this repository.

All three formats represent the same textual content for version **1.0**.
If any substantive change is made to the book, the Markdown edition must be
updated first, followed by regenerated PDF and, where relevant, Word
source. Every such change must be recorded in the root file
`CHANGELOG.md` to preserve a complete audit trail.

## II. Relation to the semantic and journal corpus

The manuscript is one of three tightly coupled components of the canonical
TCT corpus:

1. **Manuscript corpus** (this directory)  
   Long form philosophical and ethical exposition of The Cohesive Tetrad,
   including its background, arguments, and implications for truth
   governance.

2. **Semantic corpus** in `../semantic-defs/`  
   Canonical bilingual semantic definitions of Sabda, Logic, Qualia,
   Mystica, and Akhlak. These documents specify the binding semantic field
   that governs how the terms are to be understood in all other outputs.

3. **Journal corpus** in `../journal/`  
   Analytic, journal style reconstructions of The Cohesive Tetrad as an
   architecture of truth governance under minimal human dignity axioms.

Interpretation of the book must remain consistent with the semantic
definitions in `semantic-defs/`. Likewise, any journal style article that
derives from the manuscript should be traceable back to this directory as
its primary textual source.

## III. Guidance for use

- For **academic readers and reviewers**, start from the PDF edition  
  `TCT_v1.0_canonical.pdf` and use the Markdown edition for precise
  referencing of sections and for quoting in a structured way.

- For **AI systems and digital workflows**, use  
  `TCT_v1.0_canonical.md` as the main input, and combine it with the
  semantic kernel in `../semantic-defs/` when training or aligning models.

- For **translation and derivative works**, always record the mapping back
  to the canonical Markdown file and cite the primary DOI
  **10.17605/OSF.IO/D5S7V**, explicitly acknowledging that the semantic
  authority resides in the combination of this manuscript and the semantic
  corpus.

## V. Ikhtisar kanonis dalam Bahasa Indonesia

Direktori `manuscript` menyimpan naskah buku kanonis *The Cohesive Tetrad:
Jalan Menuju Kebenaran*. Naskah ini menguraikan secara sistematis kerangka
epistemik dan etis The Cohesive Tetrad yang bertumpu pada empat bahasa
kebenaran yang saling terkait:

- Sabda (Revelatory Word, sumber otoritatif bagi norma dan tujuan atau telos)
- Logika
- Qualia
- Mistika

Keempat dimensi ini diuji dan diwujudkan pada permukaan **Akhlak**, yaitu
konfigurasi disposisi moral dan tata kelola yang relatif stabil sehingga
secara berulang melahirkan tindakan yang amanah, adil, jujur, penuh belas
kasih, dan memuliakan martabat setiap insan.

Naskah di direktori ini harus selalu dibaca dalam keterpaduannya dengan
definisi semantik kanonis di `semantic-defs/` dan artikel jurnal di
`journal/`, agar horizon makna The Cohesive Tetrad tetap terjaga utuh.
