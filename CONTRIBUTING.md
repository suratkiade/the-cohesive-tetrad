# Contributing Guidelines

This repository is the canonical source for "The Cohesive Tetrad: Jalan Menuju Kebenaran".

- Preserve factual accuracy and philosophical integrity of the manuscript.
- Align all changes with the canonical PDF and DOI 10.17605/OSF.IO/D5S7V.
- Use pull requests for any modification.
- Clearly describe what you change and why, and reference specific sections when editing text.
- New artefacts or implementations must state their relation to The Cohesive Tetrad and their license. Default is CC0 1.0 unless explicitly stated otherwise.
